## Теория 

### Функция print

`print("привет")` - print(‘Привет’) - печатает на экран значения переданные в аргументе и переводит печать на следующую строку.

Строки текста должны обязательно быть экранированы кавычками ‘одинарными’ или “двойными”.

`print` - это самый простой способ взаимодействия с пользователем. Его используют в основном в консольных программах или во время отладки, чтобы быстро вывести результат.

Функция print может принимать любое число аргументов.
Чтобы передать несколько аргументов в функцию - их нужно перечислить через запятую. В результате на экран будут выведены все аргументы и между ними будут стоять пробелы.

Это еще не все возможности функции print, но для наших задач этого пока достаточно.


### Числа и действия с ними

Целые числа записываются в программе так же как на математике: 3 100 2025 -30 100500 -505

Числа с десятичной дробью записываются через точку:
1.5 0.001 177.5 -4.75

Действия с числами:
| Синтаксис          | Действие             | Результат |
|--------------------|----------------------|-----------|
| `print(2 + 3)`       | Сложение             | 5         |
|` print(50 - 65)`     | Вычитание            | -15       |
| `print(3.1 * 3) `    | Умножение            | 12,4      |
|` print(100 / 8)`     | Деление              | 12,5      |
|` print(2 ** 4)  `    | Возведение в степень | 16        |
| `print(7 // 3)  `    | Деление нацело       | 2         |
|` print(7 % 3)`       | Остаток от деления   | 1         |
| `print(2 * (3 + 4))` | Приоритет действий   | 14        |

### Переменные

Переменная - это данные, имеющие имя. Данные можно получать или изменять в любой момент вызывая имя переменной.

`age = 13`

Правила именования переменных:
- Переменные должны быть написаны только на английском языке.
- Имя переменной должно отражать значение переменной.
- Имя переменной не должно быть слишком длинным.
- Нельзя использовать зарезервированные имена переменных
- Все переменные в программе должны быть написаны в одном стиле.

### Действия со строками

Сложение строк (конкатенация) - объединяет две строки в одну
```
text = 'hello' + 'world'

'helloworld'
```
Повторение строки - повторяет строку целое число раз
```
text = 'Sale!' * 5

'Sale!Sale!Sale!Sale!Sale!'
```

### Функция len

Функция len принимает один аргумент и возвращает его длину.
Если передать строку, то возвращает количество символов в строке.

```
len(‘Hello world’) = 11
```

### Специальные символы в строках

В строках можно использовать не только обычные символы, но и специальные. Вот краткий список специальных символов в Python.

1. `\n` - перевод строки
2. `\t` - табуляция
3. `\\` - символ обратного слеша

Конечно же это не полный список. Таблицу всех спецсимолов можно найти в интернете. 

### Форматирование строк

Чтобы выводить данные в приятном глазу формате (с поясненими, отступами, округлением чисел, вудущими нулями перед числом или знаком числа) в Python используются f-строки.

```
i = 1
price = 1000 / 3
print(f"{i}) {price}")

1) 333.3333333333333
```

Чтобы отформатировать строку нужно перед строкой поставить символ `f`. Далее внутри этой строки можно в фигуных скобках ввести названия переменных и они будут подставлены в эту строку.

```
name = 'Daniel'
print(f"hello my name is {name}")

hello my name is Daniel
```

Переменные можно не просто выводить в сыром виде, но и выводить в нужном формате. Очень много форматов вывода доступно для чисел, например:

- `f'{value:.2f}'` - :.2f - округляет число до 2 знаков после запятой
- `f'{value:.0f}'` - :.0f - округляет число до 0 знаков после запятой, то есть до целого числа
- `f'{value:e}'` - :e - выводит число в формате 1.23e+09
- `f'{value:+f}'` - :+f - всегда выводит знак числа + или -

Существует очень много разных вариантов форматирования. [Ссылка для ознакомления](https://pythonworld.ru/osnovy/formatirovanie-strok-metod-format.html)
